NPM Commands

npm init: Used to initialize the setup for a new or existing project. Afterwards you'll be guided through the creation of your package.json file.
npm install: Used to install dependencies to the project.

In this example npm install express is used. Express is added to the dependencies, imported and downloaded. npm start: Commands which calls a script, made in our package.json file

Nodemon app.js: Command to run nodemon. More info can be found under the tab Nodemon & fetch above.
