Intro

REST API (RESTful API) is an application programming interface that conforms tot he contraints of REST architectural style and allows us to interact with RESTful webservices.
an API is the mediator between the user/client and the resources/web services they want to get.
REST is a set of architectural constraints where API developers can implement REST in a variety of ways.
