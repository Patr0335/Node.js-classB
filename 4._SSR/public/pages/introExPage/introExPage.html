Strings

Strings are useful for holding and storing data, which can be presented in text form.
In our String1 picture below you'll see some of the intro exercises we got handed.
Adding numbers from string to float - adding decimals to our string and using string methods such as:
indexOf() - Returns the index (position) of the first occurrence of a value in a string.
charAt() - Returns the character at a specified index (position).
replace() - Searches a string for a value, or a regular expression, and returns a string where the values are replaced
toFixed() - converts a number into a string, keeping a specified number of decimals

Objects

In Javascript almost "everything" is an object.
In Javascript, all values, except primitive values, are objects.
In the exercises given for objects we learned how to retrieve the value from object by key, and simple creation of a object. we've also learned how to add or remove a property. Code snippets can be found at the bottom of the page.

Arrays

An array is a special variable, which can hold more than one value.
Array exercises such as for loops, if statements interting/removing elements and to copy an array was some of the simple challenges we had to face.
In Exercise 6 (Array's 2 picture) we got introduced to the spread syntax (...) which allows an iterable such as an array expression or string to be expanded where zero or more arguments/elements are expected.

Functions

A JavaScript function is a block of code designed to perform a particular task.
A JavaScript function is executed when "something" invokes it (calls it).
One of the most important lessons in functions was the use of the callback function:
Callback function is a function passed as an argument to another function. Notes and example can be found in functions picture below.

Variables

Variables are containers for storing data (storing data values). Always use Let or Const.
If you want a general rule: always declare variables with const.
If you think the value of the variable can change, use let.
In the Variables exercises we were also taught why var is not allowed and how to initialize variables.
