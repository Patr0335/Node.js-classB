Nodemon

Nodemon is a tool that helps develop node.js applications by constantly refreshing/restarting your application when we change the code and files.
Requires a: npm install -g nodemon which will install it globally to the system path.
Can also be installed as a development dependency:
npm install --save-dev nodemon
The standard method was:
Node app.js
Where as Nodemon is:
Nodemon app.js

- Can write a fetch request in the front-end by heart.
Simple setup for a fetch to get data from json file:
Express serves app.js from my public folder which executes servingHTML.js. My Express route reads the template.json file located under the folder fetchExample. The client does an Ajax request via fetch(), hits the /json route which then serves the json content to our browser.
If you type in: http://localhost:9000/json you should be able to see the content of the file.
